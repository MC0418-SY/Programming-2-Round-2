Atomic Data Types
- aggregates related information of different data types

Arrays
- aggregates similar data types


struct denotes the struct function
```
struct student {
  int id;
  char name[30];
  char sex;
  int age;
}

```

struct student is a new data type

ATOMIC DATA TYPE EXAMPLE:
```
int number;
number = 50;
scanf("%d", &number);
```

CALLING STRUCTURES:
// I will call struct student
```
struct student stud1;
```

TYPEDEF:
-  It is used to create an additional name (alias) for another data type.

```
typedef struct {
  int id;
int age;
char name[30];
char sex;
}
```

STORING INFO IN STRUCTURES:
```
struct student stud1;
Student stud2;

// for direct access use:
stud1.id = 1440057;
stud2.id = 231057;
strcpy(stud1.name, "Alex");
```
